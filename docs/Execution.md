# Execution

## System-wide execution

For a system-wide setup, simply run:

```bash
$ cd foodner/
$ poetry shell
$ python cli.py dataset --help
$ python cli.py training --help
$ python cli.py evaluation --help
$ python cli.py prediction --help
```
